cd breakout-game
4. Open the project in your favorite code editor:
code .

## Technologies Used
    JavaScript
    Canvas
    CSS
## Usage
    1. Open the index.html file in your web browser.
    2. The game will load, and you can start playing.
    3. Use the left and right arrow keys to move the paddle and prevent the ball from falling to the bottom.
    4. Break all the bricks by hitting them with the ball to win.
